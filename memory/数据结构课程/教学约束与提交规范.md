## 4. 代码规范（C89）
- 语言标准：C89（C90）。
- 结构：头文件包含最小化；函数原型在前；避免可变长数组与C99特性。
- 安全：参数校验、越界检查、malloc/free 成对；返回值判错。
- 可读性：命名清晰、注释简洁；演示程序可独立编译运行。
